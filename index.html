Como exportar do seu banco
🏦 Banco do Brasil / Bradesco / Itaú

App → Extrato → Período → Exportar → CSV ou OFX. Renomeie para .csv se necessário.

🧡 Nubank / Inter / C6 Bank

App → Perfil → Extrato / Fatura → Exportar PDF ou CSV. Fatura do cartão também é aceita.

🏗️ Caixa Econômica Federal

App Caixa Tem ou site → Financiamento → Demonstrativo de pagamentos → PDF. Importe o PDF anual.

🧾 Nota Paraná

nota.pr.gov.br → Minha Conta → Notas Fiscais → Exportar planilha (XLS/CSV). Selecione o ano 2025.
